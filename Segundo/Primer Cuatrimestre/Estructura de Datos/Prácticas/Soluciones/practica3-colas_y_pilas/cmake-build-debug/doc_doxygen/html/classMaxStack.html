Detailed Description

T.D.A MaxStack.

Una instancia del tipo de dato abstracto MaxStack nos permite almacenar y manipular una secuencia de estructuras del tipo element.

El TDA MaxStack proporciona distintas herramientas para insertar, eliminar y consultar los elementos en la pila sin perder nunca de vista el mayor valor contenido en cada momento.
