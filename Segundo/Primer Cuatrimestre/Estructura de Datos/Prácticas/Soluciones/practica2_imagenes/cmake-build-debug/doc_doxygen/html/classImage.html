Detailed Description

T.D.A. Imagen.

Una instancia del tipo de dato abstracto Imagen nos permite almacenar imágenes de intensidades.

El TDA Imagen proporciona además distintas herramientas para la manipulación de dichas imágenes.

Para poder usar el TDA Imagen se debe incluir el fichero

#include <Imagen.h>

Una imagen digital puede verse como una matriz bidimensional de píxeles en la que cada uno tiene asociado un byte (unsigned char) cuyo valor está entre 0 y 255; este valor representa:

  • en imágenes en escala de grises: el nivel de luminosidad (0=negro, 255=blanco)
  • en imágenes en color: su código de color (representación por tabla de color) o la representación del color en el esquema usado (RGB, IHV, etc.).
